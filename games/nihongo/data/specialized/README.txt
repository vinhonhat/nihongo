Gói từ vựng chuyên ngành Nihongo

Cách dùng:
1. Nếu muốn load từng ngành: nhúng file JS tương ứng, ví dụ /data/specialized/construction.js
2. Nếu muốn load tất cả: nhúng all_specialized.js
3. Mỗi ngành nằm trong window.NIHONGO_SPECIALIZED_DATA.<field>
